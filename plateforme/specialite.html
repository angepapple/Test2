La Licence DAS forme les apprenants à la conception et au développement d’applications (Web, Mobile) à travers les nouvelles technologies des outils de codification.

  • Intégrateur de solutions Web
  • Architecte application mobile et Web
  • Développeur d'application Mobile et Web
  • Chef de projet développement d'application Mobile et Web
  • Entrepreneur Numérique

Pour les affectés de l’Etat (formation initiale) :

  • Être titulaire d’un Baccalauréat série C, D, E, F1, F2, F3 ;
  • Etre nouveau bachelier orienté par l’Etat ;
  • Etre âgé de 23 ans maximum.

Pour les non affectés de l’Etat (formation continue) :

  • Etre titulaire d’un Baccalauréat série C, D, E, F1, F2, F3.

Formation initiale :

  • Etudiants ivoiriens : 30.000 F CFA /An
  • Etudiants UEMOA : 100.000 F CFA /An
  • Etudiants hors UEMOA : 300.000 F CFA /An

Formation continue :

  • Etudiants ivoiriens et UEMOA : 100.000 F CFA /An
  • Etudiants hors UEMOA : 300.000 F CFA /An
